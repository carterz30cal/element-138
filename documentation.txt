--- graphics ---
the game will have levels with defined tilemaps. tilemaps should use a small set of colours, to make the game feel retro.
the base game will use tilemaps with X colours.
the colours are:
BLACK: 0,0,0
WHITE: 255,255,255
BLUE: 0,0,255
RED: 255,0,0
ORANGE: 255,128,0
YELLOW: 255,255,0
LIGHT_GREEN: 120,255,0
GREEN: 0,255,0
AQUA: 0,255,255
AZURE: 0,128,255
BROWN: 102,51,0
